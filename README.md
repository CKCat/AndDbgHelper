一个简单的 IDA 插件，简化 IDA 调试 Android 应用，目前可以在 IDA9.0 中使用。

# 要求

`adb` 和 `jdb` 在环境变量中，然后在IDA中设置 debuger 为 `Remote ARM Linux/Android debugger`，执行插件设置包名即可调试。
